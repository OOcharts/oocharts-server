OOcharts Server is a simple, installable, Google Analytics proxy service which allows Google Analytics data to be shared on public dashboards.

Installation

OOcharts Server requires NodeJS and NPM to be installed on the host.

Linux

If you're installing OOcharts on a Linux host, you can use the command below to download and run an easy setup script of the latest OOcharts Server.

wget -qO- http://oocharts.github.io/oocharts-server/install/linux.sh | sh

Custom

Download an OOcharts Release. Install dependencies and run configuration script from within the project root:

npm install

sh configure.sh
